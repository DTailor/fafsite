For the key title user lowercase letter. In case it requires more than one word to be described, use underscore (_).

Data is used only when Type is Multiple Choise. Define all possible choices by a comma separated list without spaces.

For example: student,alumni,mentor,professor

This data will be saved and showed afterwords as a JSON object. Don't try to simulate JSON and write something like ["a", "b"], this trick won't work.
