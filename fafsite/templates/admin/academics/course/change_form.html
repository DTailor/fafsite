Title

Use only lowercase letters, exception make the roman numbers. Don't use digits (write like that calculus-I). For separating more words use minus (-). For very long titles, try to cut off some info (words) by keeping it understandable.

For example:

  • Subject: Calculus I / Title: calculus-I
  • Subject: Information Security / Title: information-security
  • Subject: Professional Ethics and Communication Skills / Title: professional-ethics
